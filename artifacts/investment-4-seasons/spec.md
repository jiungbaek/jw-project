# 투자 4계절 웨더캐스터 — spec.md

## 개요
사용자가 메인 페이지에서 버튼 하나를 누르면, 야후 파이낸스 시세로 오늘 기준 미국채 10년물·원달러 환율·금·WTI·주요 지수(S&P500·나스닥·코스피)의 등락을 읽어 현재 매크로 국면을 봄/여름/가을/겨울 4계절 중 하나로 판정하고, 지표별 값·등락·신호등을 표로 나열한 뒤 그 아래 판정 설명과 일반적 자산 경향 정보를 결과 페이지에 보여준다. LLM·API 키·DB·로그인 없이 완전 stateless로 동작한다.

## 범위

### 포함
- 메인 페이지: 서비스 소개 + "오늘의 계절 확인하기" CTA
- 결과 페이지: 계절 판정(봄/여름/가을/겨울 중 1개) + 지표별(미국채 10년물, 원달러, 금, WTI, S&P500, 나스닥, 코스피) 값·일간 등락률·신호등(🟢 유리/🟡 중립/🔴 불리)을 표로 나열 + 그 아래 판정 설명 + 계절별 일반 자산군(성장주/가치주/원자재/채권 등) 경향 정보
- 계절 판정은 코드 규칙: 지수 신호(상승=유리)와 매크로 신호(금리·환율·유가·금 상승=부담)를 점수화해 4계절에 매핑
- 요청마다 야후 파이낸스 시세를 새로 조회 (캐싱 없음, 재요청 시 결과가 달라질 수 있음을 허용)
- 시세 조회 실패 시 에러 안내 + 재시도 버튼

### 제외
- LLM(Gemini 등) 판정·카피 생성 — 무료 쿼터 한도와 환각(허위 수치) 리스크 때문에 제거하고 규칙 기반으로 전환 (2026-07-09 결정)
- CPI, 국고채 10년물 — 야후 파이낸스에서 제공하지 않아 제외
- 로그인, 사용자별 이력 저장, DB — stateless 데모 목적에 불필요
- 결과 캐싱 — 매 요청 새로 판정하기로 결정
- 특정 종목명·매수/매도 시그널·목표가 추천 — 자산군 단위보다 더 구체적인 실행 지시는 스코프 밖. 재미 콘텐츠 톤을 유지하기 위해 비조언 고지 문구는 넣지 않기로 결정
- SNS 공유 카드 이미지 — 3시간 스코프 밖, 시간 남으면 다음 사이클에서 검토

## 시나리오

### 1. 정상 판정 흐름
- **Given** 사용자가 메인 페이지에 접속함
- **When** "오늘의 계절 확인하기" 버튼을 클릭함
- **Then** 결과 페이지로 이동하고, 로딩 표시 후 4계절(봄/여름/가을/겨울) 중 하나의 판정 결과와 지표 표(값·등락·신호등), 그 아래 판정 설명 및 계절별 일반 자산군 경향 정보가 표시된다

성공 기준:
- [x] 버튼 클릭 → 결과 페이지로 이동하고 로딩 상태가 보인다
- [x] 판정 완료 → "봄"/"여름"/"가을"/"겨울" 중 정확히 하나의 계절명이 화면에 표시된다
- [x] 판정 완료 → 미국채 10년물, 원달러, 금, WTI, S&P500, 나스닥, 코스피 각각의 값·등락률·신호등이 표로 표시된다 (7개 행)
- [x] 판정 완료 → 지표 표 아래에 판정 설명 문구가 표시된다
- [x] 판정 완료 → 판정된 계절에 해당하는 자산군(성장주/가치주/원자재/채권 등) 경향 문구가 표시된다

### 2. 시세 조회 실패
- **Given** 사용자가 "오늘의 계절 확인하기" 버튼을 클릭해 결과 페이지로 이동함
- **When** 야후 파이낸스 시세 조회가 실패하거나 응답 형식이 올바르지 않음
- **Then** 계절 판정 대신 에러 안내 문구와 "다시 시도" 버튼이 표시된다

성공 기준:
- [x] 시세 조회 실패 → "지금 계절을 파악하기 어렵습니다" 류의 에러 안내 문구가 표시된다
- [x] 에러 상태 → "다시 시도" 버튼이 표시된다
- [x] "다시 시도" 버튼 클릭 → 시세 조회가 재시도되고 로딩 상태로 전환된다

### 3. 재방문 시 결과 변동 허용
- **Given** 사용자가 이미 한 번 판정 결과를 본 상태
- **When** 메인 페이지로 돌아가 "오늘의 계절 확인하기"를 다시 클릭함
- **Then** 새로운 시세 조회가 실행되고, 이전 판정과 다른 계절이 나와도 정상 동작으로 취급된다 (캐시된 이전 결과를 보여주지 않음)

성공 기준:
- [x] 재클릭 → 새 로딩 상태가 다시 나타난다 (이전 결과 화면이 즉시 재사용되지 않음)

## 불변 규칙

- 결과 화면에 특정 종목명, 매수/매도 시그널, 목표가는 어떤 경로로도 표시되지 않는다 — 계절별 자산군(주식/채권/금/원자재 등) 단위의 일반적 경향 서술만 허용. 모든 카피가 코드에 정적으로 정의되므로 구조적으로 보장된다
